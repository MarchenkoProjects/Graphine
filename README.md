# Graphine

Graphine is a Java framework that provides an Object/Relational Mapping (ORM) implementation by generating native JDBC
code.

It does not currently provide an implementation of the JPA specification.

## Features

- Simplified entity markup;
- Spring Data style repositories;
- Native JDBC code generation;
- Human-readable generated source code;
- NO reflection.
